📚 راهنمای کامل اتصال DNS

🪟 ویندوز (Windows)

  1. 1
    باز کردن تنظیمات شبکه:

    کلید Windows + R → ncpa.cpl → Enter

  2. 2
    انتخاب کارت شبکه:

    روی Wi-Fi یا Ethernet راست کلیک → Properties

  3. 3
    تنظیم DNS:

    Internet Protocol Version 4 (TCP/IPv4) → Properties → Use the following DNS server addresses

  4. 4
    وارد کردن DNS:

    DNS اولیه و ثانویه را از سایت کپی کرده و وارد کنید

  5. 5
    اعمال تغییرات:

    OK → OK → اتصال مجدد به اینترنت

📱 اندروید (Android)

  1. 1
    ورود به تنظیمات:

    Settings → Network & Internet → Wi-Fi

  2. 2
    انتخاب شبکه:

    روی نام Wi-Fi متصل کلیک کنید → Modify Network

  3. 3
    تنظیمات پیشرفته:

    Advanced Options → IP Settings → Static

  4. 4
    وارد کردن DNS:

    DNS 1 و DNS 2 را از سایت کپی کرده و وارد کنید

  5. 5
    ذخیره تغییرات:

    Save → اتصال مجدد به Wi-Fi

🍎 آیفون/آیپد (iOS)

  1. 1
    ورود به تنظیمات:

    Settings → Wi-Fi

  2. 2
    انتخاب شبکه:

    روی آیکون (i) کنار نام Wi-Fi متصل کلیک کنید

  3. 3
    تنظیم DNS:

    Configure DNS → Manual

  4. 4
    اضافه کردن DNS:

    Add Server → DNS اولیه و ثانویه را اضافه کنید

  5. 5
    ذخیره:

    Save → بازگشت به صفحه اصلی

📡 مودم/روتر (Router)

  1. 1
    ورود به پنل مودم:

    192.168.1.1 یا 192.168.0.1 را در مرورگر وارد کنید

  2. 2
    ورود به حساب:

    نام کاربری و رمز عبور (معمولاً admin/admin)

  3. 3
    یافتن تنظیمات DNS:

    Network → WAN → DNS Settings

  4. 4
    تغییر DNS:

    Primary و Secondary DNS را تغییر دهید

  5. 5
    اعمال و ریستارت:

    Apply/Save → Restart Router

💡 نکات مهم و عیب‌یابی

✅ نکات مفید:
  • • پس از تغییر DNS، اتصال را قطع و وصل کنید
  • • برای بهترین نتیجه، کش DNS را پاک کنید
  • • DNS های مختلف را امتحان کنید
  • • از DNS های IPv6 برای سرعت بیشتر استفاده کنید
  • • تنظیمات را در روتر اعمال کنید تا همه دستگاه‌ها تحت تأثیر قرار گیرند
⚠️ عیب‌یابی:
  • • اگر اینترنت قطع شد، DNS قبلی را برگردانید
  • • برای پاک کردن کش: ipconfig /flushdns
  • • اگر سایتی باز نشد، DNS دیگری امتحان کنید
  • • در صورت کندی، به DNS اولیه برگردید
  • • برای مشکلات مداوم با پشتیبانی تماس بگیرید
